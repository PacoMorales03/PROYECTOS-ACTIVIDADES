XML VS HTML

XML HTML
  • Sigue las reglas generales de SGML, pero con modificaciones.
  • Especifica cómo deben definirse las etiquetas a un tipo de documento.
  • Navegador: plataforma para desarrollo de aplicaciones.
  • Usa SGML para crear un nuevo lenguaje, particular para desarrollo web.
  • Aplica un conjunto limitado de etiquetas sobre un único tipo de documento.
  • Navegador: es un visor de páginas.

XML VS SGML

XML SGML
  • Sencillo de usar.
  • Trabaja con documentos bien formados, no tienen por qué ser validados.
  • Aplicaciones de bajo coste.
  • Compatibilidad con HTML.
  • Sin etiquetas opcionales.
  • Complejo de usar.
  • Los documentos deben estar validados.
  • Aplicaciones muy costosas.
  • Solo para sectores muy específicos.
  • Sin compatibilidad directa con HTML.
