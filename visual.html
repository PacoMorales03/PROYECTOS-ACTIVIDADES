Para la instalación de Visual Studio Code vamos a seguir estos pasos:

  1. Nos dirigimos a la página de descargas de Visual Studio Code y seleccionamos nuestro sistema operativo.
  2. Se nos descargará un archivo que debemos ejecutar.
  3. Al ejecutar se nos abrirá el instalador.
  4. Seguimos los pasos para su correcta instalación
  5. Abrimos Visual Studio Code

Una vez instalado lo abrimos y buscamos las extensiones que queramos instalar.
